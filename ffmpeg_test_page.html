🎵 音頻處理測試

音頻格式轉換 -f

測試FFMPEG將各種音頻格式轉換為MP3、WAV、AAC等

ffmpeg -i input.wav -c:a libmp3lame -b:a 192k output.mp3

音頻合併 concat

測試FFMPEG將多個音頻文件合併為一個

ffmpeg -f concat -safe 0 -i filelist.txt -c copy output.mp3

音頻剪切 -ss -t

測試FFMPEG按時間範圍剪切音頻

ffmpeg -i input.mp3 -ss 00:00:30 -t 00:01:00 output.mp3

音頻元數據 -metadata

測試FFMPEG讀取和修改音頻元數據

ffprobe -v quiet -print_format json -show_format input.mp3
🎬 視頻處理測試

視頻剪切 -ss -to

測試FFMPEG按時間範圍剪切視頻片段

ffmpeg -i input.mp4 -ss 00:00:10 -to 00:00:20 -c copy output.mp4

視頻縮略圖 -vframes

測試FFMPEG從視頻中提取縮略圖

ffmpeg -i input.mp4 -ss 00:00:05 -vframes 1 output.jpg

視頻轉碼 -c:v

測試FFMPEG視頻編解碼器轉換

ffmpeg -i input.mp4 -c:v libx264 -crf 23 -c:a aac output.mp4

視頻信息 ffprobe

測試FFPROBE獲取視頻詳細信息

ffprobe -v quiet -print_format json -show_format -show_streams input.mp4
⚡ 高級功能測試

圖片轉視頻 -loop

測試FFMPEG將靜態圖片轉換為視頻

ffmpeg -loop 1 -i image.jpg -c:v libx264 -t 5 -pix_fmt yuv420p output.mp4

視頻濾鏡 -vf

測試FFMPEG視頻濾鏡效果

ffmpeg -i input.mp4 -vf "scale=1280:720,brightness=0.1" output.mp4
